# Creating a personal access token for the command line

Settings > Developer settings > Personal access tokens.
Generate new token

"config": {
        "github-oauth": {
             "github.com": "xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
        }
},
